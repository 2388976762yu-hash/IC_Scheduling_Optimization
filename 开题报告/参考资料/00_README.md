## 中期考核表若表格被改坏

**平时不要跑脚本覆盖。** 你在 WPS 里改的内容以本地文件为准；脚本默认**只补空位**，不会用空白模板覆盖。

1. 需要从零恢复表格版式时（**会丢失本地修改**）才运行：  
   `python fill_midterm.py --form-only --restore-from-blank`
2. 只补缺失字段（不动已有内容）：  
   `python fill_midterm.py --form-only`
3. 空白模板对照：`学院原版空白模板/…中期考核表_学院空白模板.doc`
