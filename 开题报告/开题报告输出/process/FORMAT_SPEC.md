# 开题报告格式规范（南开商学院模板）

## 提交形式

- **必须**在 [`2120253828-喻炫琪-研究生开题报告.doc`](../2120253828-喻炫琪-研究生开题报告.doc) 表格内填写，不得另起独立版式。
- 输出文件：[`02_开题报告_提交版.doc`](../02_开题报告_提交版.doc)（自模板复制并填表）。

## 表格对应关系（Table 2）

| 行 | 填写内容 |
|----|----------|
| R3 | 学位论文题目（中英文） |
| R4 | 课题来源 |
| R5 | 预计论文工作时间 |
| R7 | 指导小组（导师） |
| **R11** | **一、选题依据**（背景+意义+综述+参考文献） |
| **R13** | **二、研究方案** |
| **R15** | **三、研究基础** |

## 标点与用语

- 中文叙述使用**全角标点**：，。；：（）“”
- 英文缩写首次出现须给出中文释义，如“离散事件仿真（Discrete Event Simulation，DES）”
- 不用 Markdown 标题符号（#、**）进入 Word 表格

## 正文引用（著者-出版年，图 2 形式）

- 1 人：`Law（2015）`
- 2 人：`Lee和Lee（2011）`；西文 `Panwalkar和Iskander（1993）`
- 3 人及以上：`Klemmt等（2011）`；西文 `Chiu等（2023）`
- 括号使用**全角**（），与中文混排时置于句末标点之前

## 参考文献列表

- 置于「选题依据」末尾，独立成段
- **按第一责任者姓氏字母序排列**（中文按汉语拼音；Lin 与 Liu 等分别排序）
- 格式：**GB/T 7714—2015** 著录
- 序号 `[1][2]…` 在排序后依次赋予（非按正文出现顺序）
- **唯一真源**：[`BIBLIOGRAPHY.yaml`](BIBLIOGRAPHY.yaml)（32 篇，2026-06-24）
- 生成方式：运行 `fill_template.py` 自动从 YAML 生成列表

### 文献类型标识

- [J] 期刊  [M] 专著  [C] 会议论文  [D] 学位论文
