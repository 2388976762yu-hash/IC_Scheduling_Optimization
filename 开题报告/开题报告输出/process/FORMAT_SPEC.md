## 标点与用语

- 中文叙述使用**全角标点**：，。；：（）“”
- 英文缩写**首次出现**须给出中文释义：`离散事件仿真（Discrete Event Simulation，DES）`；后文直接用 DES、EDD、UPH 等，**不要**反复括号注释
- 不用 Markdown 标题符号（#、**）进入 Word 表格
- **少用括号堆砌**（详见下文「括号与列举」）

### 括号与列举

- **可以/应当用括号**：术语首次英译、国标要求的文献类型标识 [J]/[M]（仅参考文献列表）
- **尽量避免**：
  - 句内连续补注：`……（A，2008；B，2015）` → 改为 `A（2008）与B（2015）指出……`
  - 把整段说明塞进括号：`（通过Model Properties接入……）` → 拆成独立分句
  - 用 `（1）（2）（3）` 在正文段落里列举 → 综述段改用「第一，……；第二，……」；研究方案模板自带的 `（1）` 条可保留
  - `（如……）` `（参见……）` 式旁注 → 写入正文
- 设备台数、实验数字等**嵌入句中**，用顿号、分号，不用括号包裹

### 实验与初步工作表述

- 开题正文**不得**使用项目组内部编号（如 EXP-001、D-004、Model3.0 文件名等）指代实验；内部编号仅保留在 `05_Documentation/experiments/` 等工作日志中
- 引用初步仿真结果时，须用**可读的中文实验名称**，并在**首次出现处**说明实验内容，例如：「因子均为1.0的基准情景」「26情景批次因子全因子网格扫描」「仅调整订单表释放顺序的对照」
- 第二节「实验方案」用「第一步、第二步…」或「拟开展…对照实验」描述**计划**，与第一节已完成的初步工作区分
- 不宜在「研究意义」等综述性段落突然抛出未定义的实验代号与具体数值；若需举例，须承接前文已定义的实验名称

## 参考文献列表

- 置于「选题依据」末尾，独立成段
- **分两部分**，顺序固定：
  1. **中文文献**：按第一责任者姓氏**汉语拼音**字母序排列，编号 `[1]` 起
  2. **外文文献**：按第一责任者姓氏**英文字母**序排列，编号接中文文献连续编号
- 两部分之间空一行，可加小标题「中文文献」「外文文献」（与学院模板一致即可）
- 格式：**GB/T 7714—2015** 著录
- **唯一真源**：[`BIBLIOGRAPHY.yaml`](BIBLIOGRAPHY.yaml)（32 篇，2026-06-24）
- 生成方式：运行 `fill_template.py` 自动从 YAML 生成列表（**手改 Word 后勿随意重跑脚本**）
