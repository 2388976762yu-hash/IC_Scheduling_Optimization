# 开题报告格式规范（排版与著录真源）

> **写作策略**（人称、括号、实验表述、禁写项）→ 唯一见 [`WRITING_RULES.md`](WRITING_RULES.md)。  
> 本文件只管**版式、表格映射、引用与 GB/T 7714**，不另立策略条文。

## 提交形式

- **必须**在 [`2120253828-喻炫琪-研究生开题报告.doc`](../2120253828-喻炫琪-研究生开题报告.doc) 表格内填写，不得另起独立版式。
- 输出：[`02_开题报告_提交版.doc`](../02_开题报告_提交版.doc)（自模板复制；脚本只写 R11/R13/R15 文字）。
- 学院要求：Word、**宋体小四号**、A4 双面、左侧装订（见学院填写说明）。

## 表头与脚本范围

详见 [`METADATA_POLICY.md`](METADATA_POLICY.md)。摘要：

- 表头真源：模板 doc；`fill_template.py` **仅**写 R11/R13/R15；不硬编码个人信息。

## Word 单元格排版（脚本默认 = 你在 WPS 手调的目标）

| 项 | 设置 |
|----|------|
| 中文字体 | **宋体** |
| 西文字体 | **Times New Roman** |
| 字号 | **小四（12 磅）** |
| 行距 | **固定值 24 磅** |
| 段前 / 段后 | **0 行** |
| 首行缩进 | **2 字符**（24 磅） |
| 对齐 | **两端对齐** |
| **空行** | **禁止**：`1.` 与 `1.1` 之间、标题与首段之间、段与段之间均**不要空行** |

- 正文源 `section_content.py` 经 `_compact_spacing()` 去掉空行后再写入。
- 参考文献块同样**不得**插入空行（中文文献与外文文献之间亦不加空行）。
- 写 doc 后运行 `python audit_spacing.py --word`，要求 **empty_paragraphs=0**。
- **手调 02 后**：优先在 WPS 直接改；勿跑 `fill_template.py --from-template`（会从模板重建并清空版式）。
- 仅 bulk 改字时：`python fill_template.py`（默认**就地更新** 02，保留表头）；脚本仍会用上表格式重设 R11/R13/R15 段落标记。
